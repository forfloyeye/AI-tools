# 电商智能工具箱 — 产品需求文档（PRD）

> 版本：v1.1 | 更新日期：2026-03-12 | 以当前已审核前端代码为准

---

## 一、产品概述

| 项目 | 描述 |
|---|---|
| **产品定位** | 专为电商卖家设计的 Web 端轻量级 AI 图片处理工具箱 |
| **核心理念** | 即用即走、极简现代化、单页状态切换（工具内上传与编辑无跳转白屏） |
| **当前激活功能** | 智能抠图（10点）、AI 商品图（30点） |
| **规划中功能** | AI商品套图、图片翻译、服装换色、变清晰、AI扩图、AI带货视频生成 |

---

## 二、全局公共模块

### 2.1 页面布局（Layout）

- 整体背景：浅灰（`slate-50`）+ 多个装饰性高斯模糊色球（indigo / violet / fuchsia），不可交互
- 固定顶部 Navbar + 滚动主内容区

### 2.2 顶部导航栏（Navbar）

| 区域 | 内容 |
|---|---|
| **左侧** | Logo 图标（渐变 indigo→violet）+ 产品名称 `E-Tool`，点击跳转 `/` |
| **中部导航**（桌面端显示） | `一键铺货`（点击提示"开发中"）、`商品管理`（点击提示"开发中"）、`图片处理`（当前高亮，下划线指示器） |
| **右侧（未登录）** | `登录 / 注册` 按钮 → 主动唤起身份认证弹窗（可关闭） |
| **右侧（已登录）** | 🪙 `XXX 点` 积分标签 + 用户头像（点击展开下拉菜单） |

**用户头像下拉菜单**：
- 显示当前登录邮箱（截断）
- `退出登录` 按钮（红色文字）

### 2.3 统一身份认证弹窗（AuthModal）

#### 触发方式
| 场景 | 可关闭 |
|---|---|
| 主动触发：点击导航栏"登录/注册" | ✅ 可关闭（点击遮罩、点击右上角 X 均可） |
| 被动拦截：未登录点击工具卡片 | ❌ 不可关闭（遮罩点击无效，无 X 按钮） |

#### UI 结构
- 居中模态弹窗，背景半透明 + `backdrop-blur`
- 顶部两个可切换 Tab：`登录` / `注册`

#### 登录 Tab
| 元素 | 说明 |
|---|---|
| 邮箱输入框 | 校验：不能为空 + 格式合法 |
| 密码输入框 | 校验：不能为空 |
| 忘记密码链接 | 占位链接，点击无跳转（待实现） |
| `确认登录` 按钮 | 触发提交逻辑 |

#### 注册 Tab
| 元素 | 说明 |
|---|---|
| 邮箱输入框 | 校验：格式合法 |
| 密码输入框（`placeholder`：至少6位） | 校验：不能为空 + 长度 ≥ 6 |
| 确认密码输入框 | 校验：两次密码必须一致 |
| `立即注册` 按钮 | 触发提交逻辑 |

#### 表单校验规则
- 校验失败：输入框边框变红，下方显示红色提示文字
- 校验通过后进入 Loading 状态：按钮文字变为"正在处理..." + 旋转图标，所有输入框变为 Disabled

#### 提交成功流程（Mock，1.5秒延迟）
1. Loading 结束 → 调用 `login(email)`
2. 账户设置 **300 点**初始积分
3. 弹窗自动关闭
4. Navbar 右侧切换为已登录状态
5. 顶部弹出 Toast："欢迎回来！"

### 2.4 全局 Toast 提示
- 右下角或顶部弹出，3 秒后自动消失
- 类型：`success`（绿/正常色）/ `error`（红色）

### 3.2 智能抠图工作台（路由：`/tools/remove-bg`）

页面分为**左侧/中央画布区** + **右侧控制面板**（成功后激活）。

#### 状态机：`idle` → `processing` → `success`

**状态 1：`idle`（初始上传视图）**
- 全画布拖拽上传区（Dropzone）
- 文案："点击或拖拽商品图片至此"
- 支持格式提示："支持 JPG, PNG 格式，最大 10MB"
- `accept="image/*"`（前端不限制具体格式）

**状态 2：`processing`（处理中视图）**
- 原图以 30% 透明度显示为背景
- 扫描线动画（indigo 色，0%→100%→0% 循环）
- 中央白色圆角提示条："正在精准提取商品主体..."（含旋转图标）
- Mock 延迟：2000ms

**状态 3：`success`（结果+编辑视图）**
- 中央画布展示处理结果图
- 右侧控制面板解锁（从灰暗 opacity-30 变为完全可交互）

**右侧控制面板**（`success` 状态下完全可见）：

| 模块 | 内容 |
|---|---|
| **尺寸预设** | 11 个预设（见下方详表） |
| **背景底色** | 透明网格 / 纯白 / 纯黑 |
| **下载按钮** | `⬇️ 下载高清原图`（触发浏览器下载 PNG） |
| **重置按钮** | `🔄 上传新图片`（清空状态，回到 idle） |

页面左上角有 `←` 返回工作台按钮。

---

### 3.3 AI 商品图工作台（路由：`/tools/ai-scene`）

页面分为**左侧场景选择区** + **中央画布区** + **右侧控制面板**（成功后出现）。

#### 预设场景（共 5 个）

| ID | 名称 | 封面图来源 |
|---|---|---|
| `minimal` | 极简桌面 | Unsplash |
| `nature` | 自然户外 | Unsplash |
| `cozy` | 温馨家居 | Unsplash |
| `studio` | 影棚光影 | Unsplash |
| `marble` | 大理石台 | Unsplash |

#### 状态机：`idle` → `processing` → `success`

**状态 1：`idle`（初始上传视图）**
- 右侧上传区（Dropzone）
- **防呆拦截**：未选场景时尝试上传/点击 → 上传区执行抖动动画（shake）+ 显示警告："请先在左侧选择一个生成场景"
- 选中场景后：显示正常上传文案

**状态 2：`processing`（处理中视图）**
- 左侧场景选择区变为 opacity-50，不可交互
- 右侧显示所选场景封面图淡显为背景
- 中央提示："正在将商品融合至「[场景名]」..."（含旋转图标）
- Mock 延迟：2500ms

**状态 3：`success`（结果+编辑视图）**
- 中央以场景图为背景、商品图居中叠加展示合成效果
- 右侧控制面板从右侧滑入显示

**右侧控制面板**（`success` 状态）：

| 模块 | 内容 |
|---|---|
| **尺寸预设** | 11 个预设（同抠图工具，以紫色高亮） |
| **下载按钮** | `⬇️ 下载高清原图` |
| **重置按钮** | `🔄 生成其它商品`（清空图片，回到 idle，**保留场景选中状态**） |

> ⚠️ **注意**：AI 商品图工具**没有背景底色切换功能**（因为场景图自带背景，不适用纯色底）。

**成功后切换场景**：在 `success` 状态下点击左侧其他场景，自动重新触发 `processing`（再扣 30 点）。

页面左上角有 `←` 返回工作台按钮。

## 五、积分系统（当前实现）

| 规则 | 说明 |
|---|---|
| 登录赠送 | 每次登录赋予 **300 点**积分 |
| 扣费时机 | Mock 处理完成后扣费（前端状态减法） |
| 点数不足 | 显示 Toast 错误："点数不足，请充值"，自动跳回工作台大厅（`/`）可选择其他操作 |
| 充值 | 待实现（卡密充值系统规划中） |

> 📌 **后续规划**：将前端 Mock 积分替换为后端原子操作 + 卡密充值系统（详见开发文档）。
